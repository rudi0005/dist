Doomsday Button

Use what you have learned about pseudo-elements and the transform, transition and animation properties to warn the users that they are about to press the doomsday button.

  1. Add an animation to the doomsday button to warn people that it is a doomsday button. The animation should repeat forever.
  2. Warn people that they are hovering over the doomsday button by adding a different animation to the button. At least two CSS properties should be animated. The animation should repeat forever.
  3. Warn people that they are pressing the doomsday button with an even more dramatic animation. This animation should be significantly different than the first two.
  4. Extra Challenge: Using CSS animation to make the Doomsday Device go off if the button is pressed for too long.
